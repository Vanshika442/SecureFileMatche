# SecureFileMatcher

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SecureFileMatcher.git
   ```

2. Navigate to the project folder:
   ```bash
   cd SecureFileMatcher
   ```

3. Run the application:
   ```bash
   .\run.bat
   ```

The server will start and the GUI will open in your browser.

## Note
Do not close the server window until you are finished using the application.
